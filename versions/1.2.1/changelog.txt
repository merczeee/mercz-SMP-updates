Mercz SMP v1.2.1 Changelog

Added:
- Added Curios API
  - Accessory/equipment API used by many mods.
  - Helps with compatibility for mods that use extra equipment slots.

- Added Farsight
  - Allows smoother long-distance chunk viewing/caching.
  - Added new Farsight config.
  - Max render distance set to 64.
  - Chunk preview is disabled by default.
