# Mercz SMP v1.4 Changelog

## Major Additions

* Added Refined Storage support for advanced digital storage and automation.
* Expanded the Create mod ecosystem with new Create addons, machines, processing options, and automation features.
* Added new exploration content with additional structures and improved ocean travel.
* Added quality-of-life features for portable crafting, XP storage, and decorative ambience.

## Added Mods / Resources

* Refined Storage
* Refined Storage Addons
* Create Crafts & Additions
* Create Stuff 'N Additions
* Create: Enchantment Industry
* Create: Ultimate Factory
* Create: Misc & Things
* KubeJS Create
* Small Ships
* Integrated Dungeons and Structures
* Chimes
* XP Tome
* Crafting on a Stick

## Updated Mods

* No existing CurseForge-listed mods were updated in this version.

## Config / Quality-of-Life Changes

* Updated Modpack Update Checker from v1.3 to v1.4.
* Updated the in-game modpack info screen to show v1.4.
* Added new generated configs for newly added mods.
* Expanded automation and storage progression options.
* Improved exploration variety with new structures and ship-based travel.
* Added convenience features for crafting and XP management.

## Removed

* No CurseForge-listed mods were removed in this update.

## Notes

* This update mainly expands the pack with Refined Storage, more Create content, new exploration features, and several quality-of-life additions.
* Existing worlds should remain compatible, but making a backup before updating is recommended.
